**\[Basket\]** The delivery cost and its description can now be configured directly in the SCAYLE Panel.

For implementation details, refer to our [Storefront Resource Center](https://scayle.dev/en/storefront-guide/developer-guide/features/pages/basket#delivery-costs).

You can access this data in your application by using the `useShopConfigCustomData.ts` composable.
